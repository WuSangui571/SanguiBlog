# SanguiBlog V2.1.287 发布说明

> **版本性质**：稳定修复版（相对 `V2.1.285`）  
> **文档生成日期**：2026-01-08  
> **站点版本号来源**：后端 `SanguiBlog-server/src/main/resources/application.yaml` 的 `site.version`，前端首页 Banner 展示 `SANGUI BLOG // <version>`  
> **说明**：本版本聚焦后台仪表盘稳定性、暗色可读性与移动端登录输入体验；更细粒度的变更记录请参考 `.ai/CHANGELOG_AI.md`。

## 1. 版本亮点

- **后台仪表盘稳定性修复**：访客走势图 Tooltip 对齐日柱显示，且修复 Hooks 顺序错误导致的白屏问题。
- **暗色模式可读性提升**：未登录访问 `/admin` 的提示文字、登录页取消按钮在暗色模式下更清晰。
- **移动端登录输入修复**：用户名输入支持组合输入与全角转半角归一化，解决移动端双倍字符与误提示问题。

## 2. 主要变更（摘要）

### 2.1 后台仪表盘（/admin）
- 访客走势图 Tooltip 位置按日期柱中心对齐，并做边界夹取，避免固定在同一位置。
- 修复因 Hooks 调用顺序变化导致的 TrendChart 渲染崩溃与白屏。

### 2.2 登录体验（/login）
- 暗色模式下“取消”按钮文本与边框更易识别。
- 用户名/密码输入兼容移动端输入法组合输入与全角字符，避免重复字符与误报提示。

### 2.3 版本展示与脚本
- 修复版本号显示异常问题，保证首页 Banner 与 README 版本一致。
- 版本更新脚本逻辑修正（仅在需要时生成 release 文档）。

## 3. 数据库变更

本版本无数据库结构变更。

## 4. 升级与部署建议

1. **前端**：重新构建并部署 `SanguiBlog-front` 产物。
2. **后端**：本次改动为前端为主，后端无需特殊迁移。
3. **验证建议**：
   - 访问 `/admin`，悬浮访客走势图日期，Tooltip 与日期对齐且不再白屏。
   - 暗色模式下访问 `/admin` 未登录提示与 `/login` 取消按钮可读。
   - 移动端输入用户名不再出现双倍字符与误提示。

## 5. 已知注意事项

- 如控制台出现浏览器扩展注入脚本的报错（如 `content_script.js`），多为扩展噪音，与站点功能无关。
